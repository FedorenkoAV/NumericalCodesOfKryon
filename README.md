# NumericalCodesOfKryon
Приложение для вычисления числовых кодов Крайона.
Пример:
[Пиво] -> 17 10 3 16, один миллион семьсот десять тысяч триста шестнадцать -> 755-407.
(Это 1-й вариант: слово "пиво" написано в квадратных скобках и в расчётах не учитывается. Смысл этого расчёта заключается в том, что идея "пиво" проявлена только в ментальном плане, на уровне мысли, ума, и находится на уровне чисел, но в физической реальности пока не воплотилась.)

2-й вариант — это не что иное, как формула "1+1", когда, образно говоря, мысль реализуется в физическом плане, а значит, числовой код меняется, то есть к числу 755 добавляется число 46 (код слова "пиво"). В результате, запись такова:
Пиво +> 17 10 3 16, один миллион семьсот десять тысяч триста шестнадцать -> 801-189.

Таким образом, результат ображается так:
[пиво] -> 755-407
Пиво +> 801-189
(где знаки "[ ]" и "->" означает, что слово "пиво" не учитывается, а "+>" — учитывается)
